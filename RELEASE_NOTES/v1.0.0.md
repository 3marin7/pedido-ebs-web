# Release v1.0.0

**Fecha:** 2026-02-06
**Tag:** v1.0.0
**Autor:** Edwin Marin

## Resumen
Lanzamiento que agrupa mejoras de responsividad y una nueva funcionalidad visual para destacar productos recientes.

## Cambios principales
- ✨ Agregar badge 'NUEVO' para productos recientes (30 días)
- ✅ Optimizar responsividad completa de `InvoiceScreen.css`
- ✅ Agregar responsividad completa a 10 componentes más
- ✅ Agregar responsividad a 7 componentes críticos adicionales
- ✅ Correcciones de responsividad en 5 componentes principales

## Notas técnicas
- Se añadió el campo `created_at` en la consulta de `productos` para permitir el ordenamiento por "Últimos Agregados" y el badge "NUEVO".
- Badge "NUEVO" configurable por `DIAS_NUEVO = 30` en `CatalogoClientes.jsx`.
- Se aplicaron media queries base: 1024px, 768px, 480px, 360px en varios archivos CSS.
- No se realizaron cambios en la API ni en la base de datos (solo consultas). Si se requiere backfill de `created_at`, ejecutar migración.

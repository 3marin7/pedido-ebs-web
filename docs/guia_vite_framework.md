# FORMATO GUÍA DE APRENDIZAJE

## 1. IDENTIFICACIÓN DE LA GUÍA DE APRENDIZAJE

- **Denominación del Programa de Formación:** Tecnólogo en Análisis y Desarrollo de Sistemas de Información
- **Código del Programa de Formación:** 228106

```js
// Ejemplo de código de programa (main.js)
import { createApp } from 'vue';
import App from './App.vue';

createApp(App).mount('#app');
```
- **Nombre del Proyecto Formativo (si aplica):** Desarrollo de Aplicaciones Web Modernas
- **Fase del Proyecto (si aplica):** Implementación
- **Actividad de Proyecto Formativo (si aplica):** Creación de aplicaciones web con frameworks modernos usando Vite
- **Competencia:** Desarrollar aplicaciones web eficientes utilizando frameworks y herramientas modernas
- **Resultados de Aprendizaje:**
  1. **Comprender el funcionamiento de Vite como herramienta de desarrollo**
  2. **Crear proyectos web usando frameworks modernos (React, Vue, Svelte, etc.) con Vite**
  3. **Aplicar buenas prácticas en la estructura y despliegue de aplicaciones web**
- **Duración de la Guía de Aprendizaje (horas):** 8

## 2. PRESENTACIÓN

Esta guía te permitirá adquirir habilidades en el uso de Vite y frameworks modernos para el desarrollo de aplicaciones web rápidas y eficientes. A través de actividades prácticas y colaborativas, aprenderás a crear, estructurar y desplegar proyectos web, integrando conocimientos previos y promoviendo el aprendizaje autónomo y significativo.

## 3. FORMULACIÓN DE LAS ACTIVIDADES DE APRENDIZAJE

### 3.1 Actividades de reflexión inicial:
- **Descripción de la actividad:** Conversatorio sobre la evolución de las herramientas de desarrollo frontend y la importancia de los frameworks modernos.
- **Ambiente requerido:** Aula de clase o virtual.
- **Estrategias o técnicas didácticas activas:** Lluvia de ideas, discusión guiada.
- **Materiales de formación:** Pizarra, computador.
- **Material de apoyo:** Videos introductorios sobre frameworks y Vite.
- **Duración de la actividad:** 1 hora.

### 3.2 Actividades de contextualización e identificación de conocimientos necesarios para el aprendizaje:
- **Descripción de la actividad:** Presentación de conceptos clave sobre Vite, frameworks modernos y su ecosistema.
- **Ambiente requerido:** Aula de informática.
- **Estrategias o técnicas didácticas activas:** Exposición, preguntas y respuestas.
- **Materiales de formación:** Presentación digital, computador.
- **Material de apoyo:** Documentación oficial de Vite y frameworks seleccionados.
- **Duración de la actividad:** 1 hora.

### 3.3 Actividades de apropiación:
- **Descripción de la actividad:** Taller práctico de creación de un proyecto web con Vite y un framework moderno (React, Vue, Svelte, etc.).
- **Ambiente requerido:** Laboratorio de informática.
- **Estrategias o técnicas didácticas activas:** Aprendizaje basado en problemas, trabajo colaborativo.
- **Materiales de formación:** Computadores con Node.js y Vite instalados.
- **Material de apoyo:** Guía paso a paso de instalación y configuración.
- **Evidencias de aprendizaje:** Proyecto web funcional creado con Vite y framework seleccionado.
- **Instrumentos de evaluación:** Lista de chequeo, revisión de ejercicios prácticos.
- **Duración de la actividad:** 4 horas.

### 3.4 Actividades de Transferencia del Conocimiento:
- **Descripción de la actividad:** Desarrollo de un mini-proyecto donde se aplique Vite y un framework moderno para resolver una necesidad real.
- **Ambiente requerido:** Laboratorio de informática.
- **Estrategias o técnicas didácticas activas:** Aprendizaje por proyectos, exposición de resultados.
- **Materiales de formación:** Computadores, acceso a internet.
- **Material de apoyo:** Documentación de Vite y el framework utilizado.
- **Evidencias de aprendizaje:** Presentación del mini-proyecto y aplicación web funcional.
- **Instrumentos de evaluación:** Rúbrica de proyecto, presentación oral.
- **Duración de la actividad:** 2 horas.

## 5. GLOSARIO DE TÉRMINOS

- **Vite:** Herramienta de desarrollo frontend que permite crear proyectos web de manera rápida y eficiente.
- **Framework:** Conjunto de herramientas y librerías para facilitar el desarrollo de aplicaciones.
- **Frontend:** Parte visual y de interacción de una aplicación web.
